# Offline RL And Imitation Slot

Place offline or imitation-learning implementations here.

Common choices:

- behavior cloning,
- DAgger,
- GAIL/AIRL,
- CQL,
- IQL,
- BCQ,
- BEAR,
- TD3+BC,
- AWAC,
- decision transformers.

Before using offline methods, document:

- logged data source,
- coverage of relevant state-action regions,
- behavior policy if known,
- train/validation/test split,
- off-policy evaluation plan,
- distribution-shift risk,
- whether online fine-tuning is allowed.
