# Classification And Ranking Algorithm Slot

Fill this folder when the target is a class, probability, ordered preference, or score.

Recommended candidates:

- logistic regression and calibrated linear models,
- tree ensembles and gradient boosting,
- neural classifiers for high-dimensional inputs,
- pairwise or listwise ranking models,
- ordinal regression when class order matters,
- cost-sensitive models when false positives and false negatives have different downstream costs.

Document:

- label definition,
- class balance,
- calibration method,
- threshold or ranking policy,
- business or downstream cost matrix,
- metric set and acceptance thresholds.
