# Regression Algorithm Slot

Fill this folder when the selected target is continuous.

Recommended candidates:

- ridge, lasso, elastic net,
- GLM or GAM when target distribution or smooth structure matters,
- random forest and gradient boosted trees,
- neural MLP for large dense features,
- multi-output neural or tree models,
- quantile regression for service-level or interval decisions,
- conformal wrappers for finite-sample interval calibration.

Document:

- target transformation,
- sample weights,
- loss function,
- metric set,
- baseline comparison,
- uncertainty or interval method when needed.
