# Baseline Policy Notes

Define baselines before search.

Useful baselines:

- current operating rule,
- myopic optimization,
- nominal deterministic optimization,
- rolling-horizon policy,
- simple threshold or base-stock policy when structurally justified,
- random candidate only as diagnostic baseline,
- solver-derived policy when a simplified mathematical program is available.

Record baseline parameters, data assumptions, and evaluation seeds so comparisons are reproducible.
